# boxcrop

A minimal browser tool for cropping images to a specific output resolution.

Drop an image, position and resize the crop box, hit **Crop & Download** (or **Enter**) to get a PNG at your target size. If the selection is smaller than the output resolution, it gets upscaled.

**Features**

- Drag the crop box to move it, drag corners to resize (aspect ratio locked)
- Configurable output resolution — defaults to 1280×800
- Works entirely in the browser, no uploads, no server

**Usage**

Open https://voy.github.io/boxcrop in a browser, or clone and open `index.html` locally.
