# Type Talk

A mobile-first communication tool for talking to people who are hard of hearing. Type or speak — they read it off your screen in big letters.

## How it works

1. Type your message (or tap the mic and speak in Czech)
2. Press **Enter** or tap **Show to grandma** — the screen switches to a full-screen white display with giant text
3. Hold the phone up for them to read
4. Tap **Delete & back** (or press Enter) when they've read it — clears and returns to input

## Notes

- Speech recognition uses the Web Speech API (`cs-CZ`) — no account or API key needed
- Wake lock keeps the screen on while the display is shown
- Tested on iPhone with iOS Safari
