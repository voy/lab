# jazz-chords

Flashcard quiz for the chord voicings from the Autumn Leaves (Johnny Mercer) exercise in Bill Evans' *Portrait in Jazz*.

## How to use

Press **Space** or **Enter** to reveal notes and theory, then again to advance to the next chord. Chords are served in random order, infinitely.

## What's shown on reveal

- Note pills with interval labels (R, M3, P5, m7 …)
- Inversion badge
- Piano keyboard (B–C range) with:
  - Small grey dot — note is in the G major scale
  - White circle with finger number — chord tone (left hand, 5 = pinky, 1 = thumb)
  - Red border on circle — root note
- Plain-English theory explanation

## Chords covered

Am7 · D7 · Gmaj7 · Cmaj7 · F♯ø7 · B7 · Em
