# Close Tabs AI

A Chrome extension that lets you filter and close open tabs using natural language queries.

## Installation

1. Go to `chrome://extensions/` and enable Developer mode
2. Click "Load unpacked" and select this directory

## Usage

Open the popup with **Cmd+Shift+K** (Mac) / **Ctrl+Shift+K** (Windows/Linux), type a natural language description of the tabs you want to close (e.g. "all GitHub tabs" or "anything from yesterday"), and confirm.
